### Datenschutz

Wir (ITZBund) und GitHub (github.io) erheben keinerlei Daten über das Nutzungsverhalten (Tracking).

Für folgende Funktionalitäten können ausschließlich im lokalen Browser des Nutzenden Daten gespeichert werden:

- Theme-Auswahl: Das ausgewählte Theme wird im Browser-Speicher (Local-Storage) gespeichert.
- Dark-Modus: Die Information ob Tag/Nacht-Modus ausgewählt ist, wird im Browser-Speicher (Local-Storage) gespeichert.
- PWA: Die Website ist auch als Offline-Variante (PWA) verfügbar und kann vom Nutzenden lokal installiert werden.
